SARS eFiling Step-by-Step Guide

← Back to Calculator
1 Register for eFiling

Go to www.sarsefiling.co.za → Click "Register" → Select "Individual" → Complete form → Activate with SMS PIN.

2 Log In

Visit eFiling website → Enter username/password → Complete security verification → Access dashboard.

3 Start Tax Return

Click "Returns" → "File Return" → Select "ITR12" → Choose tax year 2025 → Click "Create".

4 Complete Sections

Fill in personal info → Income details → Deductions → Review → Submit electronically.

Important Deadlines:
• Non-provisional taxpayers: 31 October 2025
• Provisional taxpayers: 31 January 2026
